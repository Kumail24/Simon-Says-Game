# Simon-Says-Game
Arduino memory game which challenges user to replicate a sequence of flashing LEDs, with sequence getting longer each round
# Bill Of Materials
- 1x Arduino Uno
- 4x 220 ohm resistor
- 4x pushbutton
- 4x LED (4 separate colours)
